## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/Aakuz-Lusion/FUTURE_FS_01.git
```

Move into the project folder:

```bash
cd FUTURE_FS_01
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
